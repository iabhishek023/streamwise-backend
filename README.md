## рҹ“Ң Overview

StreamWise Backend is a Spring Boot REST API that powers the AI search functionality of the StreamWise movie finder application. It accepts natural language queries from users, sends them to Groq's LLaMA 3.3 70B model for intelligent movie recommendations, enriches results with TMDB data, and persists user search history in PostgreSQL.

## вҡҷпёҸ Setup & Installation

### Prerequisites
- Java 21+
- Maven 3.8+
- PostgreSQL 14+
- Groq API key вҶ’ [console.groq.com](https://console.groq.com)
- TMDB API key вҶ’ [themoviedb.org](https://www.themoviedb.org/settings/api)

### 1. Clone the repository
```bash
git clone https://github.com/iabhishek023/streamwise-backend.git
cd streamwise-backend
```

### 2. Create PostgreSQL database
```sql
CREATE DATABASE streamwise;
```

### 3. Configure `application.properties`
```properties
spring.application.name=streamwise

server.port=8081

spring.datasource.url=jdbc:postgresql://localhost:5432/streamwise
spring.datasource.username=postgres
spring.datasource.password=YOUR_PASSWORD

spring.jpa.hibernate.ddl-auto=update
spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.PostgreSQLDialect
spring.jpa.show-sql=true

groq.api.key=YOUR_GROQ_API_KEY
groq.api.url=https://api.groq.com/openai/v1/chat/completions
groq.model=llama-3.3-70b-versatile

tmdb.bearer.token=YOUR_TMDB_BEARER_TOKEN
tmdb.api.url=https://api.themoviedb.org/3

cors.allowed.origins=http://localhost:3000
```

### 4. Run the application
```bash
mvn spring-boot:run
```
